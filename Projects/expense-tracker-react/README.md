# React Expense Tracker

A React-based expense tracker with frontend-only login and local persistence.

## Features
- Login (frontend-only)
- Add income & expenses
- Toggle done status
- Delete expenses
- Dynamic balance calculation
- Empty state handling
- LocalStorage persistence

## Tech
- React
- JavaScript
- HTML
- CSS

## Concepts
- useState, useEffect
- Lifting state up
- Conditional rendering
- Derived state
