# React Expense Tracker

A React-based expense tracker with login and local data persistence.

## Features
- Frontend-only login
- Add income & expenses
- Toggle done status
- Delete expenses
- Dynamic balance calculation
- LocalStorage persistence

## Tech Stack
- React
- JavaScript
- HTML
- CSS

## Concepts Learned
- useState & useEffect
- Lifting state up
- Conditional rendering
- Derived state
